# Scatter plot display module UI

Renders a
[`bslib::card`](https://rstudio.github.io/bslib/reference/card.html)
containing a `plotly` scatter plot and a movie count.

## Usage

``` r
mod_plot_ui(id)
```

## Arguments

- id:

  Module namespace ID.

## Value

A [`bslib::card`](https://rstudio.github.io/bslib/reference/card.html).
