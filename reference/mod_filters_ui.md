# Filter inputs module UI

Renders two `bslib` cards: one for filter controls and one for axis
variable selectors. Intended for use inside a
[`bslib::sidebar()`](https://rstudio.github.io/bslib/reference/sidebar.html).

## Usage

``` r
mod_filters_ui(id)
```

## Arguments

- id:

  Module namespace ID.

## Value

A `tagList` of
[`bslib::card`](https://rstudio.github.io/bslib/reference/card.html)
elements.
